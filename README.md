### 핵심기능 설명

```
핵심 기능


1. 캣푸드 리뷰 기록
: 고양이에게 먹여본 제품을 다양한 기준으로 빠르게 기록합니다.
2. 서로의 리뷰 공유
: 제품마다 남겨진 리뷰들을 모아볼 수 있고, 고양이 계정을 서로 팔로우합니다.
3. 입맛이 비슷한 고양이 추천
: 내 고양이가 남긴 리뷰들의 평점을 기반으로
입맛이 비슷한 고양이를 찾아 유사도와 함께 보여줍니다.
```
